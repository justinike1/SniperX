## Technical Architecture

### Frontend Architecture
- **Framework:** React with TypeScript for type-safe development
- **Build Tool:** Vite for fast development and optimized production builds
- **Styling:** Tailwind CSS with custom dark theme and animations
- **UI Components:** Radix UI primitives with shadcn/ui component library
- **State Management:** TanStack Query for efficient server state management
- **Real-time Communication:** WebSocket integration for live market updates
- **Routing:** Wouter for lightweight client-side navigation
- **Progressive Web App:** Full PWA support for mobile installation

### Backend Architecture
- **Runtime:** Node.js 20 with TypeScript for robust server-side operations
- **Framework:** Express.js for RESTful API endpoints
- **Real-time:** WebSocket server for live data streaming and notifications
- **Database:** PostgreSQL with Drizzle ORM for type-safe database operations
- **Build System:** esbuild for optimized production bundling
- **Authentication:** JWT-based secure session management with bank-grade security
